Purpose

This driver provides a set of functions to manage:

  • Audio IN/OUT from STM32WBA55 point of view
  • Audio IN/OUT from STM32H5 point of view
  • I2C configuration of STM32H5
  • I2C configuration of LTC3556 and MAX9867 components

available on BLE Speaker from STMicroelectronics
