Purpose

This package contains specific applications running on BLE Speaker board and STM32WBA55G-DK or STM32WBA65I-DK discovery kits. See README.md files for more details.

This example is based on the STM32CubeWBA_v1.7.0 package and on the x-cube-audio-kit v1.3.0.

For further information about these components, see official package.

BLE_Audio_Broadcast_Speaker and BLE_Audio_PBP_Source_Multilanguage examples are being delivered as-is, not necessarily supported by ST.

Modified x-cube-audio-kit (Livetune) example is being delivered as-is, not necessarily supported by ST.

Update History

Main Changes

Initial release of BLE_Audio_Broadcast_Speaker and BLE_Audio_PBP_Source_Multilanguage applications running on respectively STM32WBA55 component of BLE_Speaker board and STM32WBA55G-DK or STM32WBA65I-DK discovery kits

Initial release of Livetune application running on STM32H573 component of BLE_Speaker board
